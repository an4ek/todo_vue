## Задачи
- Ознакомиться со структурой проекта на Vue (main.js, App.vue, компоненты).
- Переписать ранее реализованный To-Do List (HTML + JS) с использованием Vue 3.
- Разделить функциональность на компоненты.
- Использовать реактивные данные, computed-свойства и тд.
- Настроить сборку проекта с помощью Vite.
- Настроить пайплайн CI/CD в для сборки и развертывания проекта
- В качестве платформы можете использовать почти любой Git-хостинг.
- В качестве хостинга также можете использовать любую платформу
## Функциональные требования
- Присутствует список задач (to-do items).
- Каждая задача содержит:
- чекбокс для отметки выполнения;
- название задачи;
- кнопки редактирования и удаления;
- Присутствует кнопка добавления задачи;
- При нажатии на кнопку добавления открывается модальное окно с формой;
- Есть возможность фильтрации или поиска задач по названию;
- Есть возможность переключения темы (светлая / тёмная);
- Завершённые задачи перемещаются в отдельный список или вниз текущего.
## Требования к реализации
Технологический стек
- Vue 3 (Composition API)
- Vite — сборка и запуск проекта
- SASS — оформление интерфейса
- Tailwind CSS — опционально
